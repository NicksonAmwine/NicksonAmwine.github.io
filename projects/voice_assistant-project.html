GPT-based Voice Assistant for Luganda and English Speakers

Project Summary

I Led the design and implementation of a GPT-based inclusive voice assistant for Luganda and English speakers. A standalone system aimed at enhancing information accessibility for individuals with limited text input capabilities in low-resource language communities of Uganda.

Objective

The primary objective was to develop a standalone voice-operated ChatGPT-based assistant system that supports both Luganda and English languages for individuals especially those with limited text input and output capabilities, enhancing their ability to access information and supporting them to accomplish their specific tasks independently in their preferred language.

System Components

The system integrates both hardware and software components, utilizing a Raspberry Pi as the core hardware platform for processing user inputs and controlling other attached components such as the microphone, speakers etc.

Hardware

At the core of the system is a Raspberry Pi 3B+, which was integrated with APIs to handle user inputs and return feedback seemlessly. The system also incoporates a digital microphone for voice input and speakers for speech output.

Machine Learning

The system utilizes a combination of powerful AI models accessed through their respective APIs. The models integrated include Sunbird AI to perform Luganda STT, Whisper AI for English STT and TTS, Google translate for Luganda translation, and GPT for response generation. A custom TTS model was also trained and deployed to handle Luganda speech generation i.e. Luganda TTS.

The system can be activated using wake words while in sleep mode, allowing users to start interactions hands-free. The wake words for our system are "Myra Luganda" for system activation in Luganda and "Myra English" for English activation in English which makes it a passive listening device
